 🛒 E-Commerce Frontend Website

This project is a **frontend implementation of an e-commerce website** built using **React, Vite, and Tailwind CSS**.
The design is based on a **Figma UI template** and developed as part of a **Frontend Development Internship Task**.

The website showcases an online shopping interface including product categories, deals, and a modern UI layout.

🌐 Live Website

🔗 https://ecommerce-websitehafi.netlify.app/

📁 GitHub Repository

🔗 https://github.com/Hafiza-sadia/ecommerce-frontend-design

⚙️ Technologies Used

* React.js
* Vite
* Tailwind CSS
* JavaScript (ES6)
* HTML5
* CSS3
✨ Features

* Responsive desktop layout based on Figma design
* Modern navigation bar with search functionality
* Product categories section
* Deals and offers section
* Clean and reusable React components
* Styled using Tailwind CSS
* Fast deployment using Netlify
